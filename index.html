Reproducible Analysis (Python)

# Load and summarise financials
import pandas as pd

df = pd.read_csv('financials_q3.csv')
summary = df.groupby('segment').agg({'revenue':'sum','cost':'sum'})
summary['margin'] = (summary['revenue'] - summary['cost']) / summary['revenue']
print(summary)

Financial Formula

Weighted Average Cost of Capital (WACC):

\[\text{WACC} = \frac{E}{V}R_e + \frac{D}{V}R_d(1 - T_c)\]

Explain each term: E = equity, D = debt, V = E + D, Re = cost of equity, Rd = cost of debt, Tc = tax rate.
